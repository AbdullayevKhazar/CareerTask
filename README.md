# Basket Layihəsi

React və Tailwind CSS istifadə edərək hazırladığım sadə səbət tətbiqi.

## Xüsusiyyətlər
- Məhsulların səbətə əlavə edilməsi.
- Səbətdəki məhsulların sayının (count) artırılıb-azaldılması.
- Məhsulların səbətdən silinməsi.
- Ümumi məbləğin (Total Price) avtomatik hesablanması.

## Texnologiyalar
- React
- Tailwind CSS
- Vite

## Quraşdırma

Layihəni lokal mühitdə işə salmaq üçün terminalda aşağıdakı komandaları icra edin:

**1. Layihəni öz komputerinizə klonlayın:**
```bash
git clone https://github.com/AbdullayevKhazar/CareerTask.git
```
**2. Asılılıqları Yükləyin:**
```bash
npm install
```
### or
```bash
npm i
```
